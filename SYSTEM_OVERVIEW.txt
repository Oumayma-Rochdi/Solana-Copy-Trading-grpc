╔══════════════════════════════════════════════════════════════════════════════╗
║                        API ENDPOINTS (25+)                                   ║
╚══════════════════════════════════════════════════════════════════════════════╝

DATA SYNC:
  POST   /api/data/sync/start           Start collecting data
  POST   /api/data/sync/stop            Stop collecting data
  GET    /api/data/sync/stats           Get sync statistics

PRICES:
  GET    /api/data/token/:mint/price    Current price
  GET    /api/data/token/:mint/history  Price history (24h+)
  GET    /api/data/gainers              Top 20 gainers
  GET    /api/data/losers               Top 20 losers

WHALE WATCHING:
  GET    /api/data/whale-transactions            All whale trades
  GET    /api/data/token/:mint/whale-transactions Whale trades for token
  GET    /api/data/token/:mint/transactions      All transactions

LIQUIDITY:
  GET    /api/data/pools                Top liquidity pools
  GET    /api/data/token/:mint/pools    Pools containing token

HOLDERS:
  GET    /api/data/token/:mint/holders  Holder distribution analysis

ANALYSIS:
  GET    /api/data/token/:mint/volatility  Volatility analysis
  GET    /api/data/token/:mint/sentiment   Market sentiment
  GET    /api/data/stats/volume            Trading volume stats
  GET    /api/data/stats/marketcap         Market cap statistics

SEARCH:
  GET    /api/data/search?query=SOL     Search for tokens

WATCH LIST:
  POST   /api/data/watch                Add token to watch
  DELETE /api/data/watch/:mint          Remove from watch
  GET    /api/data/watch                View watch list


╔══════════════════════════════════════════════════════════════════════════════╗
║                       DATABASE SCHEMA (9 TABLES)                             ║
╚══════════════════════════════════════════════════════════════════════════════╝

solana_tokens
├── id (UUID)
├── mint ← TOKEN ADDRESS
├── symbol (SOL, USDC, etc)
├── name
└── supply, holders, decimals

solana_token_prices ← MAIN DATA TABLE
├── id
├── token_id → solana_tokens
├── price ← ACTUAL PRICE
├── market_cap
├── volume_24h
└── recorded_at (Updated every 30-60s)

solana_liquidity_pools
├── pool_id
├── pool_name (SOL/USDC)
├── token_a_mint, token_b_mint
├── liquidity_usd
└── volume_24h, apy

solana_transactions ← ON-CHAIN TRADES
├── tx_hash
├── mint
├── amount
├── is_whale_trade ← FLAG
└── timestamp

solana_token_holders
├── mint
├── total_holders
├── top_10_concentration (%)
├── whale_count

solana_market_metrics
├── sentiment_score (-100 to +100)
├── sentiment_label (bullish/bearish)
└── momentum_score

solana_orderbook_data
├── best_bid, best_ask
├── spread_percent
└── recorded_at

solana_data_sources
├── source_name (birdeye, helius, raydium)
├── last_sync
├── error_count
└── sync_frequency_seconds

solana_sync_logs
├── action (fetch_prices, fetch_liquidity, etc)
├── status (success, error)
├── records_processed
└── execution_time_ms


╔══════════════════════════════════════════════════════════════════════════════╗
║                     USE CASES & EXAMPLES                                     ║
╚══════════════════════════════════════════════════════════════════════════════╝

1. WHALE WATCHING 🐋
   • Monitor large transactions
   • Alert on $1M+ trades
   • Copy whale trades automatically
   
   Example:
   GET /api/data/whale-transactions
   → Returns: [{ token: SOL, amount: 5M, time: now, ... }]

2. TREND FOLLOWING 📈
   • Identify top gainers
   • Check volatility before trading
   • Follow momentum
   
   Example:
   GET /api/data/gainers
   → Returns: [{ symbol: "COPE", change: +45%, volume: 50M, ... }]

3. COPY TRADING 📋
   • Track specific wallets
   • Copy their positions
   • Manage risk
   
   Example:
   GET /api/data/token/{mint}/whale-transactions
   → Filter for whale address
   → Execute similar trades

4. RISK MANAGEMENT ⚠️
   • Analyze holder concentration
   • Check volatility levels
   • Assess market sentiment
   
   Example:
   GET /api/data/token/{mint}/holders
   → Check if top 10 has < 30%
   → Calculate risk score

5. LIQUIDITY OPTIMIZATION 💧
   • Find best pools
   • Analyze APY
   • Optimize swaps
   
   Example:
   GET /api/data/pools
   → Filter by liquidity > $1M
   → Sort by APY

║                      QUICK START (5 MINUTES)                                 ║
╚══════════════════════════════════════════════════════════════════════════════╝

1. SET API KEYS
   $ export BIRDEYE_API_KEY="your_key"
   $ export HELIUS_API_KEY="your_key"
   $ export DATABASE_URL="postgresql://..."

2. CREATE TABLES
   $ psql -d $DATABASE_URL -f scripts/create-solana-data-tables.sql

3. START BOT
   $ npm start

4. TEST DATA
   $ curl http://localhost:3000/api/data/gainers

5. USE IN YOUR APP
   → Add WATCH_TOKENS to .env
   → Start trading with real data!
